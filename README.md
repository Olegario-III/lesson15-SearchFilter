# Search Filter App - React

A clean React application that demonstrates real-time search filtering. It fetches a list of users from a public API and allows users to search by name dynamically.

## ✨ Features

- Fetches user data from JSONPlaceholder API
- Real-time search filtering by name
- Responsive user list display
- Shows "No users found" message when search yields no results
- Clean and minimal UI

## 🛠 Technologies Used

- **React.js** (Functional Components + Hooks)
- `useState` and `useEffect`
- Fetch API for data fetching
- JavaScript (ES6+)
- CSS (via App.css)
